Try it: CSS Spanish Listener 1.1


This app is very CSEC focused and is designed to help with the Listening Paper

If you can get all of these questions correct you will get first 15-20 questions correct in exam

Try it: Duolingo


This is an incredible app to develop basic vocabulary and listening skills in Spanish (And other languages)

It isn't CSEC focused, however getting over about 1500Xp means you are probably good enough to pass the exam.

Try it: Spanish Dict


This app contains our favourite Spanish dictionary. It has lots of translations, slang, and conjugations in various tenses.
